Digital Ricoeur: Ricoeur's Social Network

This will serve as an interactive presentation of Ricoeur's Social Network, based on teh frequency at which Ricoeur refers to other thinkers.
We currently have a static bar graph, using a CSV file with made-up data.
The CSV file comprises of two columns, Thinker and Frequency as seen in the graph.
The user may wish to be able to sort the graph or interact with it in some other way.
While sorting function will require changes in the format of the CSV file, other interactivities could be added easily.
Also, we may want to experiment with the ability to add multiple CSV files.
Another fix that may be required is dynamic sizing of the font for the x-axis, as Thinkers with longer names (e.g. Thomas Aquinas) spill over to other bars.
